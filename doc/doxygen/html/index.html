daM3System code documentation

Introduction

daM3System stands for data acquisition magnetic moments and MOKE.
This software is designed for controlling VSM and MOKE setups. Please find more abstract ducumentation here: (/doc/ppt). You need MATLab® and the Data Acquisition Toolbox™ and optional the Statistics Toolbox™ in order to compile this software.
