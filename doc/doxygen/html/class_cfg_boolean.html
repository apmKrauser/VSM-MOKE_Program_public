Detailed Description

Extends a check box in order to automatically update the corresponding Config parameter.

The Axes and Text objects can be placed using Mathworks's GUIDE(TM)

usage: guiobj = PkgAdvGUI.CfgBoolean(hObject, cfg_obj, cfg_varname) hObject Handle of text box object cfg_obj Optional; An object or structure that should get automatically updated on value changes of the PhyEditBox cfg_varname Optional; The property of cfg_obj which will be updated

The value can be changed with: guiobj.setValue(value);
