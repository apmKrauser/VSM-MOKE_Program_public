function fMPlot ( in  varargin)

Measurement, Calibration and Plot window usage:

Usage:

plotwindow = fMPlot(window_appearance, additional_windowtext);
@b additional_windowtext: additional text displayed in window title
@b window_appearance:
window_appearance = Config.mplotview_magnetcalibration : window layout for magnet calibration
window_appearance = Config.mplotview_measure : window layout for a normal measurement
window_appearance = Config.mplotview_viewcalibration : window layout for viewing a voltage-field plot of a magnet calibration file
window_appearance = Config.mplotview_viewdatafile : window layout for viewing previously measured magnetization data
Parameters
vararginInteger defined by Config.mplotview_*; specifying use case related window appeareance
Return values
varargoutnot used
